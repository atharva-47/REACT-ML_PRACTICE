# Assignment 4 - Redux Practice

Simple React app with Redux and Redux Toolkit examples.

## What's included:

- Q1: Shopping cart with Redux
- Q2: Login system with JWT tokens  
- Q3: Notes app using Redux Toolkit
- Q4: API calls with RTK Query
- Q5: Comparing old Redux vs new Redux Toolkit

## How to run:

```bash
npm install
npm run dev
```
